# Blog API Website

## Description

The Blog API Website is a robust web application developed with EJS and Express.js, enabling seamless CRUD (Create, Read, Update, Delete) operations. This project incorporates efficient methods for managing blog posts, precise post updates and deletions, and optimized search functionality to enhance data accuracy and user experience. Axios is used for smooth communication between the front-end and back-end, ensuring a seamless user experience.

## Key Features

- **Seamless CRUD Operations:** Implemented using EJS and Express.js for creating, reading, updating, and deleting blog posts.
- **Precise Post Management:** Utilized `Find` and `FindIndex` methods for accurate updates and deletions of blog posts.
- **Efficient Search Functionality:** Used the `Filter` method to enhance search performance, reducing blog retrieval time by 3 seconds.
- **Smooth Front-end to Back-end Communication:** Leveraged Axios for efficient data handling and user experience.

## Technologies Used

- **EJS:** Embedded JavaScript templating for dynamic HTML generation.
- **Express.js:** Web framework for Node.js to build the server-side application.
- **Axios:** Promise-based HTTP client for making requests to the back-end.
- **JavaScript:** For implementing core functionality and methods like `Find`, `FindIndex`, and `Filter`.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone 
    ```
2. **Navigate to the project directory:**
    ```bash
    cd blog-api-website
    ```
3. **Install the required dependencies:**
    ```bash
    npm install
    ```

## Usage

1. **Run the application:**
    ```bash
    npm start
    ```
2. **Open your browser and navigate to:**
    ```bash
    http://localhost:3000
    ```
3. **Interact with the blog:**
    - **Create** new blog posts.
    - **Read** existing blog posts.
    - **Update** posts with precise control.
    - **Delete** posts accurately.
    - **Search** for specific posts efficiently.
